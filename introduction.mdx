# Welcome to AlXuraan

AlXuraan is a modern Islamic companion mobile application built with React Native and Expo, designed to bring essential Islamic resources to your fingertips. Whether you're seeking to read the Quran with unique Wolof translations, track prayer times, or explore hadith collections, AlXuraan provides a seamless and beautiful experience.

## Key Features

### Quran Reader
- **114 Surahs** with Arabic text and translations
- **Wolof translations** from local JSON data (quran_wolof.json)
- **French translations** from Hamidullah edition
- **Audio playback** for each ayah with multiple reciters
- **Favorites system** to bookmark verses
- **Search functionality** across translations

### Prayer Times
- **Automatic location detection** using Expo Location
- **Aladhan API integration** for accurate prayer times
- **Customizable notifications** for each prayer
- **Hijri calendar** display
- **Background fetch** support for timely notifications

### Hadith Library
- Multiple hadith collections
- French translations
- Search across hadiths
- Categorized by topics
- Bookmark favorite hadiths

### Audio Player
- Background audio playback
- Queue management
- Playback controls (play, pause, seek)
- Support for multiple reciters
- Audio continues when app is backgrounded

## Technology Stack

AlXuraan is built with modern technologies:

- **React Native 0.79.4** - Cross-platform mobile framework
- **Expo SDK 53** - Development platform and tooling
- **React Navigation 7** - Navigation solution
- **NativeWind 4** - Tailwind CSS for React Native
- **Zustand** - State management
- **Expo Audio/AV** - Audio playback capabilities
- **Axios** - HTTP client for API requests

## Architecture

The app follows a clean architecture with:

```text
alxuraan/
├── screens/          # Screen components (HomeScreen, QuranScreen, etc.)
├── components/       # Reusable UI components
├── services/         # API and business logic (quranService, prayerTimesService)
├── navigation/       # Navigation configuration
├── context/          # React Context providers (ThemeContext, PlayerContext)
├── hooks/           # Custom React hooks
├── data/            # Local JSON data (Wolof translations)
└── utils/           # Utility functions
```

## API Integrations

AlXuraan integrates with multiple Islamic APIs:

- **AlQuran Cloud API** - Quran text and audio
- **Quran.com API** - Recitations and reciters
- **QuranEnc API** - French translations
- **Aladhan API** - Prayer times and Hijri calendar

## App Configuration

The app is configured via `app.config.ts` with:

```typescript
{
  name: 'alxuraan',
  slug: 'alxuraan',
  version: '1.0.0',
  ios: {
    bundleIdentifier: 'com.devdiop.alxuraan',
    supportsTablet: true,
    infoPlist: {
      UIBackgroundModes: ['audio']
    }
  },
  android: {
    package: 'com.devdiop.alxuraan'
  }
}
```
