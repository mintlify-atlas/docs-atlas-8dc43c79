## Overview

The Dhikr Counter feature provides a digital tasbih (prayer beads) for counting remembrance of Allah. It includes preset collections for the 99 Names of Allah, morning and evening adhkar, and post-prayer dhikr with customizable counters and progress tracking.

## Data Structures

### Dhikr Object

```typescript TypeScript Interface
interface Dhikr {
  id: number;
  arabic: string;              // Arabic text
  transliteration: string;     // Romanized pronunciation
  translation: string;         // French/English meaning
  repetitions: number;         // Recommended repetition count
  category?: string;           // 'morning' | 'evening' | 'prayer'
}
```

### Allah Name Object

```typescript TypeScript Interface
interface AllahName {
  id: number;                  // Name number (1-99)
  arabic: string;              // Arabic name (e.g., 'الرَّحْمَنُ')
  transliteration: string;     // Romanized (e.g., 'Ar-Rahman')
  meaning: string;             // Translation/meaning
}
```

## Counter Implementation

Typical counter UI implementation:

```javascript Counter Component
import { useState } from 'react';
import { View, TouchableOpacity, Text, Vibration } from 'react-native';

const DhikrCounter = ({ dhikr }) => {
  const [count, setCount] = useState(0);
  const target = dhikr.repetitions;

  const handlePress = () => {
    if (count < target) {
      setCount(count + 1);
      Vibration.vibrate(50); // Haptic feedback
    }
  };

  const reset = () => {
    setCount(0);
  };

  const isComplete = count >= target;

  return (
    <View>
      <Text>{dhikr.arabic}</Text>
      <Text>{dhikr.transliteration}</Text>
      <Text>{dhikr.translation}</Text>
      
      <TouchableOpacity 
        onPress={handlePress}
        disabled={isComplete}
      >
        <Text>{count} / {target}</Text>
      </TouchableOpacity>
      
      <TouchableOpacity onPress={reset}>
        <Text>Reset</Text>
      </TouchableOpacity>
    </View>
  );
};
```

### Custom Dhikr

Allow users to create custom dhikr:

```javascript Custom Dhikr
const createCustomDhikr = async (dhikr) => {
  const customDhikr = {
    id: Date.now(),
    arabic: dhikr.arabic,
    transliteration: dhikr.transliteration,
    translation: dhikr.translation,
    repetitions: dhikr.repetitions || 33,
    isCustom: true
  };

  const existing = await AsyncStorage.getItem('@custom_dhikr');
  const customList = existing ? JSON.parse(existing) : [];
  customList.push(customDhikr);
  
  await AsyncStorage.setItem('@custom_dhikr', JSON.stringify(customList));
  return customDhikr;
};
```

## Storage Strategy

### Save Counter State

```javascript Persist Counter
const saveCounterState = async (dhikrId, count) => {
  const key = `@dhikr_count_${dhikrId}`;
  await AsyncStorage.setItem(key, JSON.stringify({
    count,
    timestamp: Date.now()
  }));
};

const loadCounterState = async (dhikrId) => {
  const key = `@dhikr_count_${dhikrId}`;
  const data = await AsyncStorage.getItem(key);
  return data ? JSON.parse(data) : { count: 0 };
};
```

### Daily Statistics

```javascript Track Daily Stats
const updateDailyStats = async (category, count) => {
  const today = new Date().toISOString().split('T')[0];
  const key = `@dhikr_stats_${today}`;
  
  const existing = await AsyncStorage.getItem(key);
  const stats = existing ? JSON.parse(existing) : {};
  
  stats[category] = (stats[category] || 0) + count;
  stats.total = (stats.total || 0) + count;
  
  await AsyncStorage.setItem(key, JSON.stringify(stats));
};
```
