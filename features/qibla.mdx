## Overview

The Qibla Finder helps users determine the direction to the Kaaba in Mecca for prayer. It uses GPS location services and device magnetometer sensors to calculate the precise bearing and display a visual compass pointing toward the Qibla.

## API Functions

### Get Qibla Direction

<CodeGroup>

```javascript Calculate Qibla
// Source: services/qiblaService.jsx:13-43
getQiblaDirection: async () => {
  try {
    // Request location permission
    const { status } = await Location.requestForegroundPermissionsAsync();

    if (status !== 'granted') {
      throw new Error('Location permission not granted');
    }

    // Get current position
    const location = await Location.getCurrentPositionAsync({});
    const currentCoords = {
      latitude: location.coords.latitude,
      longitude: location.coords.longitude,
    };

    // Calculate bearing to Kaaba
    const bearing = geolib.getRhumbLineBearing(currentCoords, KAABA_COORDS);

    return {
      bearing,                                                    // Direction in degrees
      distance: geolib.getDistance(currentCoords, KAABA_COORDS) / 1000,  // km
      currentCoords,
      kaabaCoords: KAABA_COORDS,
    };
  } catch (error) {
    console.error('Error getting qibla direction:', error);
    throw error;
  }
}
```

```javascript Start Magnetometer
// Source: services/qiblaService.jsx:45-56
startMagnetometerUpdates: (callback) => {
  Magnetometer.setUpdateInterval(100); // Update every 100ms

  const subscription = Magnetometer.addListener((data) => {
    // Calculate compass angle
    const angle = Math.atan2(data.y, data.x) * (180 / Math.PI);
    callback(angle);
  });

  return subscription;
}
```

```javascript Stop Magnetometer
// Source: services/qiblaService.jsx:58-63
stopMagnetometerUpdates: (subscription) => {
  if (subscription) {
    subscription.remove();
  }
}
```

### Hook Implementation

<CodeGroup>

```javascript Full Hook Code
// Source: services/qiblaService.jsx:67-117
export const useQibla = () => {
  const [qiblaData, setQiblaData] = useState(null);
  const [compassHeading, setCompassHeading] = useState(0);
  const [subscription, setSubscription] = useState(null);
  const [loading, setLoading] = useState(true);
  const [error, setError] = useState(null);

  useEffect(() => {
    let mounted = true;

    const initQibla = async () => {
      try {
        setLoading(true);
        const data = await qiblaService.getQiblaDirection();

        if (mounted) {
          setQiblaData(data);

          // Start magnetometer
          const sub = qiblaService.startMagnetometerUpdates((heading) => {
            if (mounted) {
              setCompassHeading(heading);
            }
          });

          setSubscription(sub);
          setLoading(false);
        }
      } catch (err) {
        if (mounted) {
          setError(err.message);
          setLoading(false);
        }
      }
    };

    initQibla();

    return () => {
      mounted = false;
      if (subscription) {
        qiblaService.stopMagnetometerUpdates(subscription);
      }
    };
  }, []);

  // Calculate qibla angle relative to current direction
  const qiblaAngle = qiblaData ? (qiblaData.bearing - compassHeading + 360) % 360 : 0;

  return { qiblaData, compassHeading, qiblaAngle, loading, error };
};
```

</CodeGroup>

## Bearing Calculation

The service uses the **Rhumb Line Bearing** method for direction calculation:

<Note>
  A rhumb line (or loxodrome) is a path of constant bearing, which makes it ideal for compass navigation. While not the shortest distance, it's easier to follow with a compass.
</Note>

```javascript Bearing Formula
// Using geolib library
import * as geolib from 'geolib';

const bearing = geolib.getRhumbLineBearing(
  { latitude: userLat, longitude: userLon },
  { latitude: kaabaLat, longitude: kaabaLon }
);

// Returns: 0-360 degrees (0 = North, 90 = East, 180 = South, 270 = West)
```

### Distance Calculation

```javascript Distance Formula
// Using geolib library
const distanceInMeters = geolib.getDistance(

  { latitude: kaabaLat, longitude: kaabaLon }

const distanceInKm = distanceInMeters / 1000;
```

## Sensor Calibration

<Warning>
  Magnetometer accuracy can be affected by magnetic interference from metal objects, electronics, and buildings. Always calibrate the compass before use.
</Warning>

### Calibration Instructions

<Steps>
  <Step title="Move Away from Metal">
    Step away from large metal objects, electronics, and magnetic sources
  </Step>
  <Step title="Figure-8 Motion">
    Move the phone in a figure-8 pattern to calibrate the magnetometer
  </Step>
  <Step title="Level Surface">
    Hold the phone flat and level for best accuracy
  </Step>
  <Step title="Verify North">
    Check that the compass points to true north before using
  </Step>
</Steps>

### Calibration Component

```javascript Calibration Screen
const CalibrationScreen = ({ onComplete }) => {
  const [calibrating, setCalibrating] = useState(true);
  const [accuracy, setAccuracy] = useState(0);

  useEffect(() => {
    const subscription = Magnetometer.addListener(({ x, y, z }) => {
      // Calculate magnetic field strength
      const strength = Math.sqrt(x * x + y * y + z * z);
      
      // Estimate accuracy (simplified)
      const expectedStrength = 50; // μT (microtesla)
      const accuracyPercent = Math.min(
        100,
        (1 - Math.abs(strength - expectedStrength) / expectedStrength) * 100
      );
      
      setAccuracy(accuracyPercent);
      
      if (accuracyPercent > 80) {
        setCalibrating(false);
        setTimeout(onComplete, 1000);
      }
    });

    return () => subscription.remove();
  }, []);

  return (
    <View>
      <Text>Calibrating Compass...</Text>
      <Text>Move phone in figure-8 pattern</Text>
      <ProgressBar progress={accuracy / 100} />
    </View>
  );
};
```
