# Quick Start Guide

Get AlXuraan up and running on your development machine in just a few steps. This guide will have you viewing the app on your device or simulator quickly.

## Prerequisites

Before you begin, ensure you have:

- **Node.js 18+** installed on your machine
- **npm** or **yarn** package manager
- **Expo Go app** on your iOS/Android device (for quick testing)
- **Xcode** (for iOS development on Mac)
- **Android Studio** (for Android development)

<Note>
  For the fastest start, use the Expo Go app. For full features like background audio and notifications, you'll need a development build.
</Note>

## Installation Steps

<Steps>
  <Step title="Clone the Repository">
    Clone the AlXuraan source code to your local machine:
    
    ```bash
    git clone https://github.com/devdiop221/alxuraan.git
    cd alxuraan
    ```
  </Step>

  <Step title="Install Dependencies">
    Install all required npm packages:
    
    ```bash
    npm install
    ```
    
    Or if you prefer yarn:
    
    ```bash
    yarn install
    ```
    
    This will install all dependencies including:
    - React Native 0.79.4
    - Expo SDK 53
    - React Navigation 7
    - Expo Audio, Notifications, Location
    - NativeWind and Tailwind CSS
  </Step>

  <Step title="Start the Development Server">
    Launch the Expo development server:
    
    ```bash
    npm start
    ```
    
    Or with the dev client:
    
    ```bash
    npm run start
    ```
    
    This runs: `expo start --dev-client`
    
    You'll see a QR code in your terminal and a web interface will open in your browser.
  </Step>

  <Step title="Run on Your Device">
    Choose your platform:
    
    <Tabs>
      <Tab title="iOS">
        For iOS Simulator (requires Mac with Xcode):
        
        ```bash
        npm run ios
        ```
        
        This runs: `expo run:ios`
      </Tab>
      
      <Tab title="Android">
        For Android Emulator or connected device:
        
        ```bash
        npm run android
        ```
        
        This runs: `expo run:android`
      </Tab>
      
      <Tab title="Expo Go">
        1. Install Expo Go from App Store or Play Store
        2. Scan the QR code from your terminal
        3. The app will load on your device
        
        <Warning>
          Some features like background audio won't work in Expo Go. Use a development build for full functionality.
        </Warning>
      </Tab>
    </Tabs>
  </Step>

  <Step title="Grant Permissions">
    When the app launches, grant the following permissions:
    
    - **Location**: Required for prayer times based on your location
    - **Notifications**: Required for prayer time alerts
    - **Motion & Orientation**: Required for Qibla finder
    
    The app will request these on first launch via `App.jsx`:
    
    ```jsx
    // Location permission
    const { status } = await Location.requestForegroundPermissionsAsync();
    
    // Notification permission  
    const { status } = await Notifications.requestPermissionsAsync();
    ```
  </Step>
</Steps>

## Verify Installation

Once the app loads, you should see:

1. **Home Screen** with:
   - Current time display
   - Prayer times for your location
   - Hijri date
   - Next prayer countdown

2. **Bottom Navigation** with icons for:

   - Compass (Qibla)
   - Messages

3. **Feature Cards** for:

   - Adhan
   - Qibla

## Testing Key Features

### Prayer Times

The app fetches prayer times from the Aladhan API based on your location:

```jsx
// From HomeScreen.jsx
const locationData = await aladhanService.getUserLocation();
const times = await aladhanService.getPrayerTimes(
  locationData.latitude,
  locationData.longitude
);
```

### Quran Reading

Browse surahs and read with Wolof and French translations:

```jsx
// From quranService.jsx
const surahs = await getSuraNames();
const translation = await getSuraTranslation(suraNumber);

// Each verse includes:
// - Arabic text
// - French translation (Hamidullah)
// - Wolof translation (from local JSON)
// - Audio URL
```

### Audio Player

Play Quran recitations with background audio support:

```jsx
// Configured in app.config.ts
ios: {
  infoPlist: {
    UIBackgroundModes: ['audio']
  }
}
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="App won't start or shows errors">
    1. Clear node_modules and reinstall:
    ```bash
    rm -rf node_modules
    npm install
    ```
    
    2. Clear Expo cache:
    ```bash
    npx expo start -c
    ```
    
    3. Check Node.js version (should be 18+):
    ```bash
    node --version
    ```
  </Accordion>

  <Accordion title="Location permission denied">
    The app needs location for prayer times. Grant permission in:
    - iOS: Settings > AlXuraan > Location
    - Android: Settings > Apps > AlXuraan > Permissions
  </Accordion>

  <Accordion title="Prayer times not loading">
    1. Ensure location permission is granted
    2. Check internet connection
    3. Verify Aladhan API is accessible
    4. Check console for error messages
  </Accordion>

  <Accordion title="Audio playback not working">
    Audio features require a development build, not Expo Go:
    ```bash
    npm run build:dev
    ```
    Then install the generated build on your device.
  </Accordion>
</AccordionGroup>
