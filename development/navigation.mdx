## Overview

AlXuraan uses [React Navigation](https://reactnavigation.org/) for navigation, implementing a tab-based navigation pattern with nested stack navigators. The app features a custom-designed bottom tab bar with gradient effects and smooth transitions.

## Navigation Architecture

```
NavigationContainer
└── AppNavigator (Tab Navigator)
    ├── Récitateurs → SurashNavigator (Stack)
    ├── Coran → QuranNavigator (Stack)
    ├── Prières → PrayerTimesScreen
    ├── Dhikr → HadithNavigator (Stack)
    ├── Favoris → FavoritesScreen
    └── Paramètres → SettingsScreen
```

## Main Tab Navigator

The `AppNavigator.jsx` file defines the main tab navigation with a custom tab bar component.

<CodeGroup>

```jsx AppNavigator.jsx
import React from 'react';
import { createBottomTabNavigator } from '@react-navigation/bottom-tabs';
import { View, Platform } from 'react-native';
import { BookOpen, Clock, Radio, Settings, Star, BookMarked } from 'lucide-react-native';
import { LinearGradient } from 'expo-linear-gradient';
import { COLORS, SPACING, BORDER_RADIUS, SHADOWS } from '../components/ui/Theme';

import QuranNavigator from './QuranNavigator';
import SurashNavigator from './SurashNavigator';
import PrayerTimesScreen from '../screens/PrayerTimesScreen';
import HadithNavigator from './HadithNavigator';
import FavoritesScreen from '../screens/FavoritesScreen';
import SettingsScreen from '../screens/SettingsScreen';
import PlayerMini from '../components/PlayerMini';
import { usePlayer } from '../context/PlayerContext';

const Tab = createBottomTabNavigator();
const TAB_BAR_HEIGHT = Platform.OS === 'ios' ? 85 : 65;

export default function AppNavigator() {
  const { currentAudio, closePlayer } = usePlayer();

  return (
    <View style={{ flex: 1, backgroundColor: COLORS.background }}>
      <Tab.Navigator
        tabBar={props => <TabBar {...props} />}
        screenOptions={{
          headerShown: false,
        }}
      >
        <Tab.Screen name="Récitateurs" component={SurashNavigator} />
        <Tab.Screen name="Coran" component={QuranNavigator} />
        <Tab.Screen name="Prières" component={PrayerTimesScreen} />
        <Tab.Screen name="Dhikr" component={HadithNavigator} />
        <Tab.Screen name="Favoris" component={FavoritesScreen} />
        <Tab.Screen name="Paramètres" component={SettingsScreen} />
      </Tab.Navigator>

      {currentAudio && (
        <View style={{
          position: 'absolute',
          bottom: TAB_BAR_HEIGHT + SPACING.md * 2,
          left: 0,
          right: 0,
          zIndex: 1000,
          paddingHorizontal: SPACING.sm,
        }}>
          <PlayerMini
            audioUrl={currentAudio.audioUrl}
            surahName={currentAudio.surahName}
            surahNumber={currentAudio.surahNumber}
            onClose={closePlayer}
          />
        </View>
      )}
    </View>
  );
}
```

</CodeGroup>

<Info>
  The app integrates the `PlayerMini` component directly in the navigator, allowing it to persist across tab changes.
</Info>

## Custom Tab Bar

AlXuraan features a custom-designed tab bar with gradient backgrounds and icon animations.

```jsx Custom TabBar Component
const TabBar = ({ state, descriptors, navigation }) => {
  return (
    <View
      style={{
        position: 'absolute',
        bottom: SPACING.md,
        left: SPACING.md,
        right: SPACING.md,
      }}>
      <LinearGradient
        colors={['#1F1A2C', '#1F1A2C']}
        start={{ x: 0, y: 0 }}
        end={{ x: 1, y: 1 }}
        style={{
          flexDirection: 'row',
          height: TAB_BAR_HEIGHT,
          borderRadius: BORDER_RADIUS.xl,
          padding: SPACING.xs,
          backgroundColor: '#1F1A2C',
          borderWidth: 1,
          borderColor: '#1F1A2C',
          ...SHADOWS.large,
        }}>
        {state.routes.map((route, index) => {
          const { options } = descriptors[route.key];
          const isFocused = state.index === index;

          const onPress = () => {
            const event = navigation.emit({
              type: 'tabPress',
              target: route.key,
              canPreventDefault: true,
            });

            if (!isFocused && !event.defaultPrevented) {
              navigation.navigate(route.name);
            }
          };

          // Icon rendering logic...
          return (
            <TouchableOpacity key={route.key} onPress={onPress}>
              {/* Tab icon with gradient background when active */}
            </TouchableOpacity>
          );
        })}
      </LinearGradient>
    </View>
  );
};
```

## Stack Navigators

Each major feature has its own stack navigator for managing screen hierarchies.

### Quran Navigator

<CodeGroup>

```jsx QuranNavigator.jsx
import { createStackNavigator } from '@react-navigation/stack';
import { COLORS, SPACING } from '../components/ui/Theme';
import QuranScreen from '../screens/QuranScreen';
import SourateDetail from '../screens/SourateDetail';

const Stack = createStackNavigator();

export default function QuranNavigator() {
  return (
    <Stack.Navigator
      screenOptions={{
        cardStyle: { backgroundColor: COLORS.background },
        headerStyle: {
          backgroundColor: COLORS.cardBackground,
          elevation: 0,
          shadowOpacity: 0,
          height: 60,
        },
        headerTitleStyle: {
          color: COLORS.textPrimary,
          fontSize: 18,
          fontWeight: '600',
        },
        headerTintColor: COLORS.primary,
        headerBackTitleVisible: false,
        cardStyleInterpolator: ({ current: { progress } }) => ({
          cardStyle: {
            opacity: progress,
          },
        }),
        headerShown: false,
      }}
    >
      <Stack.Screen name="Quran" component={QuranScreen} />
      <Stack.Screen name="SourateDetail" component={SourateDetail} />
    </Stack.Navigator>
  );
}
```

</CodeGroup>

## Navigation Usage

### Navigating Between Screens

Use the `navigation` prop passed to screen components:

```jsx Example Navigation
import { useNavigation } from '@react-navigation/native';

function QarisScreen() {
  const navigation = useNavigation();

  const handleQariPress = (qari) => {
    navigation.navigate('Surahs', { 
      reciterId: qari.id,
      reciterName: qari.name 
    });
  };

  return (
    // ... component JSX
  );
}
```

### Passing Parameters

```jsx Passing Data
// Navigate with parameters
navigation.navigate('SourateDetail', {
  surahNumber: 1,
  surahName: 'Al-Fatiha'
});

// Access parameters in destination screen
function SourateDetail({ route }) {
  const { surahNumber, surahName } = route.params;
  // ... use parameters
}
```

### Going Back

```jsx Navigation Back
// Go back to previous screen
navigation.goBack();

// Check if can go back
if (navigation.canGoBack()) {
  navigation.goBack();
}
```

## Navigation State Integration

The navigation system integrates with global state through Context:

```jsx PlayerContext Integration
function AppNavigator() {
  const { currentAudio, closePlayer } = usePlayer();

  return (
    <View style={{ flex: 1 }}>
      <Tab.Navigator {...navigationConfig} />
      
      {/* Mini player persists across navigation */}
      {currentAudio && (
        <PlayerMini
          audioUrl={currentAudio.audioUrl}
          surahName={currentAudio.surahName}
          surahNumber={currentAudio.surahNumber}
          onClose={closePlayer}
        />
      )}
    </View>
  );
}
```

<Info>
  By placing the `PlayerMini` component in the `AppNavigator`, it persists across all tab and stack navigation changes.
</Info>

## Best Practices

1. **Screen Options**: Define common screen options at the navigator level
2. **Deep Linking**: Configure linking for direct navigation to specific screens
3. **Navigation Guards**: Implement authentication checks in navigation listeners
4. **Type Safety**: Use TypeScript types for navigation params
5. **Performance**: Use `React.memo()` for screen components to prevent unnecessary re-renders
6. **Header Management**: Keep headers consistent across related screens

## Common Navigation Patterns

### Modal Navigation

```jsx Modal Screen
<Stack.Screen 
  name="PlayerScreen" 
  component={PlayerScreen}
  options={{
    presentation: 'modal',
    headerShown: true,
  }}
/>
```

### Reset Navigation Stack

```jsx Reset Stack
navigation.reset({
  index: 0,
  routes: [{ name: 'Home' }],
});
```

### Navigate to Nested Screen

```jsx Nested Navigation
navigation.navigate('Récitateurs', {
  screen: 'Player',
  params: { surahId: 1 },
});
```

<Note>
  Always use the `navigation` prop or `useNavigation()` hook instead of storing navigation references in variables.
</Note>
