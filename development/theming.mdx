## Overview

AlXuraan uses a centralized theming system built with React Native and NativeWind (Tailwind CSS for React Native). The theme is defined in `components/ui/Theme.jsx` and provides a consistent design language across the entire application.

## Theme Context

AlXuraan uses React Context to manage theme state across the application.

### ThemeProvider

The theme provider is located at `context/ThemeContext.jsx`:

```jsx context/ThemeContext.jsx
import React, { createContext, useContext, useState } from 'react';

export const ThemeContext = createContext();

export const ThemeProvider = ({ children }) => {
  const [theme, setTheme] = useState('light');

  const toggleTheme = () => {
    setTheme((prevTheme) => (prevTheme === 'light' ? 'dark' : 'light'));
  };

  return (
    <ThemeContext.Provider value={{ theme, toggleTheme }}>
      {children}
    </ThemeContext.Provider>
  );
};

export const useTheme = () => useContext(ThemeContext);
```

### Usage

Wrap your app with the `ThemeProvider`:

```jsx App.jsx
import { ThemeProvider } from './context/ThemeContext';
import AppNavigator from './navigation/AppNavigator';

export default function App() {
  return (
    <ThemeProvider>
      <AppNavigator />
    </ThemeProvider>
  );
}
```

Access theme in components:

```jsx
import { useTheme } from '../context/ThemeContext';

function MyComponent() {
  const { theme, toggleTheme } = useTheme();
  
  return (
    <TouchableOpacity onPress={toggleTheme}>
      <Text>Current theme: {theme}</Text>
    </TouchableOpacity>
  );
}
```

## Complete Example

Here's a complete example using the theming system:

```jsx SettingsScreen.jsx
import React from 'react';
import { View, TouchableOpacity, ScrollView } from 'react-native';
import { LinearGradient } from 'expo-linear-gradient';
import { Settings } from 'lucide-react-native';
import { CustomText } from '../components/ui/Typography';
import { 
  COLORS, 
  SPACING, 
  BORDER_RADIUS, 
  GRADIENTS, 
  SHADOWS 
} from '../components/ui/Theme';

export default function SettingsScreen() {
  return (
    <LinearGradient colors={GRADIENTS.primary} style={{ flex: 1 }}>
      <View style={{ padding: SPACING.md }}>
        <View style={{
          backgroundColor: COLORS.cardBackground,
          borderRadius: BORDER_RADIUS.lg,
          padding: SPACING.lg,
          marginBottom: SPACING.md,
          ...SHADOWS.base,
        }}>
          <View style={{
            flexDirection: 'row',
            alignItems: 'center',
            marginBottom: SPACING.sm,
          }}>
            <View style={{
              width: 40,
              height: 40,
              borderRadius: BORDER_RADIUS.full,
              backgroundColor: COLORS.buttonBg,
              justifyContent: 'center',
              alignItems: 'center',
              marginRight: SPACING.sm,
            }}>
              <Settings size={20} color={COLORS.textPrimary} />
            </View>
            <CustomText weight="bold" color={COLORS.textPrimary}>
              Settings
            </CustomText>
          </View>
        </View>
      </View>
    </LinearGradient>
  );
}
```

<Note>
  All theme values are exported from `components/ui/Theme.jsx` for easy import and consistent usage across the application.
</Note>

## Best Practices

1. **Always use theme constants** - Never hardcode colors, spacing, or other design values
2. **Use CustomText component** - For consistent typography across the app
3. **Apply shadows via spread operator** - Use `...SHADOWS.base` to apply shadow styles
4. **Leverage LinearGradient** - Use predefined gradient arrays for consistent backgrounds
5. **Keep accessibility in mind** - Ensure sufficient color contrast for text readability

## NativeWind Integration

AlXuraan also uses NativeWind for Tailwind-style utility classes:

```jsx

  </Text>
```
